# 崩坏学园2 自动刷奖励关脚本

> 用于自动刷《崩坏学园2》阵营战等奖励关卡  
> 支持自动补充体力、选择是否使用双倍券、是否切换备用装备  
> 相比模拟器自带连点器，可避免因游戏卡顿或网络延迟导致的点击时间错乱问题

---

## ✨ 功能特点

- ✅ 自动识别窗口并锁定游戏区域  
- ✅ 基于图像模板匹配进行点击，适应不同界面状态  
- ✅ 自动检测并补充体力（支持购买/兑换）  
- ✅ 支持是否启用“双倍券”  
- ✅ 支持是否切换备用装备  
- ✅ 随机点击偏移，降低被检测风险  
- ✅ 鼠标移至屏幕角落可紧急中断（PyAutoGUI 安全机制）  

---

## 🖥 运行环境

- Windows 操作系统  
- [Mumu 模拟器]  
- 崩坏学园2 游戏已运行并进入对应奖励关卡界面  
- Python 3.10 

---

运行以下命令安装所需 Python 库：

```bash
pip install pyautogui opencv-python numpy pillow pywin32
```

## 🧩 脚本配置说明

### 1. 图片模板准备

脚本依赖以下图片模板（请放在 `test/` 文件夹中）：

| 文件名 | 说明 |
|--------|------|
| `beiyong.png` | 备用装备按钮 |
| `kaizhan.png` | 开战按钮 |
| `qie.png` | 切备用按钮 |
| `tiaozhan_diyala.png` | 再次挑战按钮（示例命名） |
| `shuangbeiquan_diyala.png` | 双倍券按钮 |
| `buchong.png` | 补充体力按钮 |
| `goumai.png` | 购买体力按钮 |
| `duihuan.png` | 兑换体力按钮 |

> 不同角色/界面建议重新截图并修改脚本中的 `tiaozhan` / `shuangbeiquan` 变量路径

### 2. 脚本变量调整

```python
tiaozhan = "test/tiaozhan_diyala.png"
shuangbeiquan = "test/shuangbeiquan_diyala.png"
```

根据你截图的角色或界面风格修改文件名。

---

## 🚀 使用步骤

1. 启动 MuMu 模拟器并进入崩坏学园2  
2. 进入你要刷的奖励关卡界面（例如阵营战）  
3. 运行脚本：

```bash
python jiaoben.py
```

4. 根据提示：
   - 是否切换备用装备（输入 `1` 或 `0`）  
   - 是否使用双倍券（输入 `1` 或 `0`）  
5. 用鼠标点击游戏窗口（脚本会自动锁定该窗口并显示红框）  
6. 脚本开始自动循环刷关  

> ⚠️ 运行期间不要移动或最小化游戏窗口

---

## 🛡 紧急中断方式

将鼠标 **快速移动到屏幕任意角落**（四个角之一），脚本会立即停止。

## ⚠️ 注意事项

- 脚本基于窗口绝对坐标 + 图像匹配，请保持模拟器窗口大小与截图时一致  
- 如果游戏界面出现较大变化（如UI更新），需重新截图模板  
- 建议在模拟器设置中关闭“窗口置顶”以外的干扰项  
- 不支持后台运行，游戏窗口必须可见且不被遮挡
